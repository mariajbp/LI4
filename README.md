## Server

### Installing

```shell
python3 -m venv ticketnow-env

# Windows
ticketnow-env\Scripts\activate.bat

# Unix / MacOS
source ticketnow-env/bin/activate

pip install -r requirements.txt
```

### Executing

```shell
# Preferentially with super user permitions
python3 src/app.py
```
